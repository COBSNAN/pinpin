# pinpin

## 介绍

拼单的轻社交平台。

## 技术
后端运用 nodejs express 框架
前端运用 bootstrap 实现PC端和移动端的适配。
数据库 运用的是mysql 数据库

nodejs 用到的模块

- mysql
- multer
- moment
- emojione
- ejs
- socket.io
